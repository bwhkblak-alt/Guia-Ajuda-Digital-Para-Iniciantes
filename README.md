Guia de Ajuda Digital para Iniciantes

Este projeto tem como objetivo auxiliar pessoas que possuem dificuldade em usar o celular e acessar serviços digitais básicos.
A proposta é criar um guia simples e acessível, com orientações claras sobre como realizar tarefas comuns do dia a dia.

Objetivo

Promover inclusão digital, facilitando a vida de pessoas que não têm familiaridade com tecnologia.

Conteúdos abordados

Como usar o WhatsApp

Como instalar aplicativos

Como acessar a conta gov.br

Como enviar fotos e vídeos

Como aumentar a letra e usar recursos de acessibilidade

Como usar a câmera e galeria


Tecnologias envolvidas

Este repositório utiliza apenas documentação simples em markdown (README.md), ideal para iniciantes.
