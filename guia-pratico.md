Guia Prático de Ajuda Digital para Iniciantes

Este guia foi criado para auxiliar pessoas que têm dificuldade em usar o celular, aplicativos e serviços digitais básicos. O objetivo é oferecer orientações simples e detalhadas para que qualquer pessoa, mesmo sem experiência, consiga aprender no seu próprio ritmo.


---

1. Entendendo o Celular

1.1. Botões principais

Botão de ligar/desligar: localizado na lateral. Pressione e segure para ligar ou desligar o aparelho.

Botões de volume: aumentam ou diminuem o som.

Tela inicial: é a primeira tela após desbloquear o celular.


1.2. Como desbloquear

Pressione o botão de ligar uma vez.

Deslize o dedo de baixo para cima.

Se houver senha, digite o código numérico solicitado.


1.3. Movimentos básicos

Toque rápido: usado para abrir aplicativos.

Segurar o dedo: usado para mover, excluir ou configurar itens.

Deslizar o dedo: usado para rolar a tela ou navegar entre páginas.



---

2. Como Usar o WhatsApp

O WhatsApp é o principal meio de comunicação digital e uma das ferramentas mais importantes para iniciantes.

2.1. Como abrir o WhatsApp

1. Localize o ícone do WhatsApp na tela inicial ou na lista de aplicativos.


2. Toque uma vez para abrir.



2.2. Como enviar uma mensagem de texto

1. Abra o WhatsApp.


2. Toque no nome do contato desejado.


3. Toque na caixa onde aparece “Digite uma mensagem”.


4. Escreva o texto.


5. Toque no botão de enviar.



2.3. Como enviar uma mensagem de áudio

1. Abra a conversa com o contato.


2. Pressione e segure o botão de gravação.


3. Fale a mensagem.


4. Solte o botão para enviar.



2.4. Como enviar fotos que já estão na galeria

1. Abra a conversa com o contato.


2. Toque no botão de anexar.


3. Selecione “Galeria”.


4. Escolha a foto que deseja enviar.


5. Toque em “Enviar”.



2.5. Como tirar uma foto e enviar na hora

1. Abra a conversa com o contato.


2. Toque no botão de câmera.


3. Tire a foto.


4. Toque em “Enviar”.




---

3. Como Instalar Aplicativos

3.1. Acessando a Play Store

1. Abra a Play Store no celular.


2. Toque na barra de pesquisa e escreva o nome do aplicativo desejado.


3. Toque em “Instalar”.


4. Aguarde o download e a instalação.


5. O aplicativo estará disponível na lista de apps.




---

4. Como Tirar e Enviar Fotos

4.1. Como tirar uma foto

1. Abra o aplicativo da câmera.


2. Aponte para o que deseja fotografar.


3. Toque no botão de captura.



4.2. Como enviar pelo WhatsApp

1. Abra o WhatsApp.


2. Entre no contato desejado.


3. Toque em “Galeria”.


4. Selecione a foto.


5. Toque em “Enviar”.




---

5. Como Acessar Serviços Públicos

O principal serviço utilizado no Brasil é o aplicativo gov.br.

5.1. Como entrar no gov.br

1. Abra o aplicativo gov.br.


2. Digite o CPF.


3. Digite a senha cadastrada.


4. Acesse o serviço desejado, como consulta de benefícios, documentos digitais ou cadastro.




---

6. Segurança Digital

6.1. Cuidados essenciais

Não clicar em links enviados por desconhecidos.

Não compartilhar senhas.

Desconfiar de pedidos de dinheiro.

Evitar enviar documentos pessoais sem confirmação.

Atualizar sempre os aplicativos para maior segurança.
